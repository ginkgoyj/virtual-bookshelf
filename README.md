# Claude书架

一个单页静态展示网页，以虚拟书架的形式呈现书籍和物件信息，包含书脊布局、详情浮层和自定义视觉样式。

## 功能

- 以书架形式展示书籍和物件
- 按两层书架分布内容
- 点击书脊或摆件打开详情浮层
- 在详情中展示标题、作者、摘录和说明文字
- 支持遮罩点击或 `Esc` 关闭浮层
- 使用同一份数据驱动页面渲染

## 技术栈

- HTML
- CSS
- 原生 JavaScript
- SVG 内嵌图形

## 开发环境

- 现代浏览器

## 项目结构

- 主页面：`index.html`

## 核心实现

### UI 结构

页面由两层书架、书脊元素、摆件元素和详情浮层组成。整体视觉采用深色背景、暖色灯光、书脊阴影和封面细节。

### CSS 表现

样式中包含：

- 书架层次与阴影
- 书脊高光与立体感
- 浮层背景模糊
- 详情页封面、标题和说明区排版

### JavaScript 逻辑

页面数据集中定义在 `ITEMS` 数组中，每个条目包含类型、位置、尺寸、颜色和详情文本。脚本负责：

- 渲染两层书架内容
- 处理点击后的详情加载
- 管理当前选中状态
- 关闭详情浮层

### 内容类型

除了书籍，页面还支持沙漏、纸条等物件条目，并为不同类型使用不同展示逻辑。
